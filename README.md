# Capillary Mat Lab

植物育成向けの 3D プリント製毛細管マットを、ブラウザ上で設計して STL として書き出す静的 Web アプリです。

## Features

- パラメータ変更に応じたリアルタイム形状生成
- `縦筒アレイ`, `ジグザグ横孔`, `交互ラダー孔`, `蛇行ウィック`, `葉脈ネット`, `gyroid土壌` のベータ比較モード
- 従来の `旧格子案` と `旧層状案` も比較用に残存
- WebGL ベースの 3D プレビュー
- 平面図プレビュー
- 共有 URL の生成
- ブラウザ内での ASCII STL 書き出し

## Parameters

### 縦筒アレイ

- 幅 / 長さ / 厚み
- 外周フレーム幅
- 鉛直毛細管の内径
- X/Y 方向の外壁クリアランス
- 上面播種カップ径 / 深さ

### ジグザグ横孔

- `縦筒アレイ` の全パラメータ
- 横孔の内径
- 横孔の高さ率

### 交互ラダー孔

- `縦筒アレイ` の全パラメータ
- 連通孔の内径
- 上段連通孔の高さ率

### 蛇行ウィック

- 幅 / 長さ / 厚み
- 1 本の連続ウィックの空隙幅
- 隣接壁の最小厚み
- 走路どうしの空気ギャップ
- 外周フレーム幅
- 折り返し部の播種ポケット有無と径

### 葉脈ネット

- 幅 / 長さ / 厚み
- 主脈の空隙幅
- 側脈の空隙幅
- 側脈ピッチ
- 側脈の伸び長さ
- 枝先ポケット有無と径
- 外周フレーム幅

### gyroid土壌

- 幅 / 長さ / 厚み
- 外周フレーム幅
- 基本セル径
- 骨格の目標厚み
- 厚み方向の伸長率

### 旧比較モード

- 旧格子案: 毛細管幅 / 壁厚 / X/Y 密度 / 交点ディンプル
- 旧層状案: 層ラティス開口 / リブ厚 / レイヤー数 / 縦ウィック密度

## Beta Concepts

- `縦筒アレイ`: 鉛直円柱毛細管を素直に並べるベースライン案
- `ジグザグ横孔`: 円柱毛細管どうしを上寄りのジグザグ横孔でつなぐ通気強化案
- `交互ラダー孔`: 隣接円柱毛細管を上下 2 段の横孔で疎につなぐ 3D 寄り案
- `蛇行ウィック`: 1 本の連続 path を蛇行させる、最も単純で検証しやすい基準案
- `葉脈ネット`: 中央主脈から側脈へ分岐させる、有機的な分配ネットワーク案
- `gyroid土壌`: 3D プリントの積層特性を活かした TPMS 系スポンジ案

## Legacy Parameters

- 旧格子案 / 旧層状案は、以前の検討条件を比較するために残しています

### 旧層状案

- 幅 / 長さ / 厚み
- 外周フレーム幅
- スポンジ胞の開口
- 層ラティスのリブ厚
- 積層レイヤー数
- 層ごとのオフセット率
- 縦ウィック径
- 縦ウィック間隔

## Local Preview

静的ファイルなので、任意の HTTP サーバーでそのまま配信できます。

```bash
python3 -m http.server 8000
```

その後 `http://localhost:8000` を開いてください。

## Deploy

このリポジトリは GitHub Pages の `main` ブランチ / root 配信を前提にしています。
